# Tkrell Branding Guidelines

**Last Updated:** November 7, 2025

## Brand Identity

Tkrell is an AI-powered educational platform for Kenya that combines the best of IXL's mastery learning, SaveMyExams' comprehensive resources, Duolingo's engaging gamification, and Eneza's accessibility. Our brand reflects **innovation, warmth, motivation, and inclusivity**.

## Brand Values

- **Empowerment:** We help students take control of their learning journey
- **Accessibility:** Education for all, from KG to IGCSE, online and offline
- **Innovation:** AI-powered personalized learning with Mr. T
- **Community:** Building a supportive network of learners across Kenya
- **Excellence:** Striving for academic success through smart, engaging tools

## Visual Identity

### Color Palette

Our color scheme is inspired by Kenya's vibrant culture and the psychology of learning:

#### Primary Colors

**Teal (#00BFA6)**
- **Usage:** Primary brand color, buttons, links, accents
- **Psychology:** Trust, growth, intelligence, calmness
- **OKLCH:** `oklch(0.68 0.15 180)`

**Coral (#FF6B6B)**
- **Usage:** Secondary color, highlights, call-to-action, achievements
- **Psychology:** Energy, encouragement, warmth, motivation
- **OKLCH:** `oklch(0.72 0.18 25)`

**Cream (#FFF8E1)**
- **Usage:** Background, cards, soft surfaces
- **Psychology:** Comfort, clarity, focus, approachability
- **OKLCH:** `oklch(0.97 0.03 90)`

#### Supporting Colors

- **Success Green:** For correct answers, achievements
- **Warning Yellow:** For streaks, alerts, attention
- **Error Red:** For incorrect answers, warnings
- **Neutral Gray:** For text, borders, subtle elements

### Typography

#### Headings
- **Font Family:** System font stack (sans-serif)
- **Weight:** Bold (700) for primary headings, Semibold (600) for subheadings
- **Style:** Clean, modern, readable

#### Body Text
- **Font Family:** System font stack (sans-serif)
- **Weight:** Regular (400) for body, Medium (500) for emphasis
- **Line Height:** 1.5-1.6 for readability

## Design Principles

### 1. Gamification and Motivation

**Visual Feedback:**
- Confetti animations on perfect scores
- Progress bars with vibrant colors
- Badge and achievement displays
- Streak counters with fire emoji üî•

**Micro-Interactions:**
- Button hover effects
- Loading animations
- Smooth transitions
- Celebratory messages ("Nice one! +5 XP!")

### 2. Psychology of Engagement

**Dopamine Triggers:**
- Instant feedback on quiz submissions
- Visible XP and coin rewards
- Level-up notifications
- Leaderboard rankings

**Fear of Missing Out (FOMO):**
- Streak counters encourage daily use
- Limited-time challenges (future feature)
- Social leaderboards

**Social Validation:**
- County and school-based rankings
- Referral system rewards
- Teacher-student classroom connections

### 3. Accessibility and Inclusivity

**Readability:**
- High contrast text (WCAG AA compliant)
- Clear font sizes (minimum 14px for body text)
- Generous line spacing

**Responsive Design:**
- Mobile-first approach
- Touch-friendly buttons (minimum 44px tap targets)
- Adaptive layouts for all screen sizes

**Language:**
- Simple, encouraging language
- Kenyan English expressions where appropriate
- Clear instructions and error messages

## UI Components

### Cards
- **Border Radius:** 1rem (16px) for soft, friendly appearance
- **Shadow:** Subtle shadows for depth
- **Hover Effects:** Border color change, slight elevation

### Buttons
- **Primary:** Teal background, white text, rounded
- **Secondary:** Coral background, dark text, rounded
- **Outline:** Transparent background, teal border
- **Sizes:** Small (sm), Medium (default), Large (lg)

### Forms
- **Input Fields:** Rounded corners, clear labels, helpful placeholders
- **Validation:** Inline error messages in red, success in green
- **Focus States:** Teal ring around active inputs

### Notifications
- **Success:** Green background, checkmark icon
- **Error:** Red background, alert icon
- **Info:** Blue background, info icon
- **Warning:** Yellow background, warning icon

## Voice and Tone

## Application Examples

### Landing Page
- Hero section with gradient background (teal to coral)
- Large, bold headline with brand promise
- Feature cards with icons and descriptions
- Social proof (student count, testimonials)
- Clear call-to-action buttons

### Dashboard
- Gamification stats at the top (Level, Streak, Coins, XP)
- Quick action cards for main features
- Recent activity feed
- Progress visualizations

### Quiz Interface
- Clean, distraction-free question display
- Clear answer options with radio buttons
- Progress indicator
- Encouraging feedback after submission

### Chat with Mr. T
- Chat bubble interface (user on right, Mr. T on left)
- Markdown support for formatted responses
- Suggested prompts for new users
- Friendly avatar or icon for Mr. T

## Brand Applications

### Marketing Materials
- Use primary teal and coral prominently
- Include student success stories
- Highlight AI and gamification features
- Emphasize "Free for 1 Year" offer

### Social Media
- Consistent color scheme across platforms
- Share study tips, motivational quotes
- Celebrate student achievements
- Use hashtags like #LearnWithTkrell #MrTutor

### Email Communications
- Branded header with logo
- Warm, encouraging tone
- Clear call-to-action buttons
- Footer with legal links
